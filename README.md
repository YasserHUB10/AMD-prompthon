# Smart Food & Health Assistant 🌱

A lightweight, fully responsive, zero-dependency web application to log meals, track calories, and receive AI-based meal suggestions tailored to your goals. 

## Features
- **Food Logger**: Quickly add meals and calorie counts. Data is persisted in your browser via `localStorage`.
- **Health Dashboard**: View your total calories, meals logged, and a dynamically calculated Health Score.
- **Goal Selector**: Choose between Weight Loss, Maintain, or Weight Gain.
- **AI Meal Suggestions**: Rule-based logic provides food recommendations specific to your selected fitness goal.
- **Offline First**: All data is managed locally, ensuring instant load times and complete privacy.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla - ES6+)
- *No frameworks, no backend required.*
- **Storage**: `localStorage` native API.

## Setup Instructions

Simply open `index.html` in any modern web browser! No installation, dependencies, or building required.

## Demo Flow
1. **Choose Your Goal**: Select a goal (e.g., Weight Loss, Gain).
2. **AI Action**: See the AI recommendations instantly adapt below.
3. **Log a meal**: E.g., "Oatmeal with Berries" for 300 calories.
4. **Live Dashboard**: Watch the Dashboard instantly update your calorie sum, meal count, and algorithmic Health Score.
